macOS package

Run packaging/macos/package.sh to create dist/Personal-Chess-Tutor-macOS.zip. The app contains the backend, CLI, browser assets, local catalogs, and tactical-corpus manifest. Stockfish remains an explicit free local dependency.

First run

  1. Install Stockfish with brew install stockfish, or set PCT_STOCKFISH.
  2. Move the app to Applications and open it.
  3. The launcher stores data under ~/Library/Application Support/Personal Chess Tutor and opens the loopback UI.

Data operations

Use the packaged Contents/Resources/bin/pct-data command for location, backup, restore, reset, and uninstall guidance. Restore first preserves the current directory with a timestamp instead of deleting it.

Qualification

scripts/clean-install-smoke.sh

The smoke test packages from scratch, launches with an isolated temporary data directory, checks loopback health, restarts, and checks diagnostics.
