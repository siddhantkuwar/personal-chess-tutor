Release evidence

Demo

The scripted demo uses a synthetic game, an isolated data directory, and local Stockfish.

Run

scripts/demo.sh
# Open http://127.0.0.1:8787

Walkthrough

  1. Import the synthetic Italian Game PGN.
  2. Navigate while shallow and deep updates arrive.
  3. Open a ranked mistake and its evidence.
  4. Start a drill, fail once, reveal an earned hint, and retry.
  5. Inspect profile denominators, trends, and recommendations.
  6. Batch import and demonstrate reserved interactive capacity.
  7. Restart with the same directory and show replayed results.

The package smoke automates the nonvisual release path: clean package, loopback launch, sample PGN import, completed analysis, restart, diagnostics, and persisted-game reopen. A narrated video can be recorded from the same demo script without changing release behavior.
