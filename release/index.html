A reproducible map from product claims to code, tests, measurements, and packaging. The tutor remains local, telemetry-free, and independent of paid services.

Runtime

Prioritized bounded game jobs feed an isolated Stockfish pool. Two workers are the conservative default; historical work may use only one, reserving interactive capacity.

Correctness

Array and bitboard models share a compile-time board concept and pass parity, make/unmake, hash, FEN, legal-move, and perft checks.

Reliability

Append, snapshot, compaction, engine, queue, cancellation, restart, and concurrent load paths have explicit fault or endurance coverage.

Release

The macOS packager bundles native binaries, web assets, catalogs, data tools, and a manifest-tracked optional tactical corpus.
