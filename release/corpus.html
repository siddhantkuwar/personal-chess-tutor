Optional tactical corpus

The integration targets the official Lichess puzzle export, published under Creative Commons CC0. The application ships only a tiny curated manifest; a full download is optional and one-time.

Download and select

curl -O https://database.lichess.org/lichess_db_puzzle.csv.zst
unzstd lichess_db_puzzle.csv.zst
node scripts/import-lichess-puzzles.js \
  --input lichess_db_puzzle.csv \
  --output "$HOME/Library/Application Support/Personal Chess Tutor/lichess-corpus.json" \
  --version YYYY-MM-DD --limit 5000 --min-rating 600 --max-rating 2400

The converter streams the large CSV, caps output at 50,000 rows, filters low-popularity and out-of-range puzzles, maps supported motifs, records license/source/version, and refuses to overwrite an existing output file.

Use

personal-chess-tutor --tactical-corpus /absolute/path/to/lichess-corpus.json

Validation gates

  1. Apply the corpus setup move.
  2. Parse the resulting FEN and require a legal solution move.
  3. Match a recurring personal motif and rating within 400 points.
  4. Prefer personal-game drills for that category.
  5. Require two fresh local engines to select the solution.
  6. Reject alternatives within 50 centipawns.
  7. Persist provenance and validation evidence.

Transformed positions are intentionally disabled.
